# Curriculo Web - Guilherme Andraz

Landing page do curriculo com foco em frontend. Projeto estatico feito com HTML, CSS e JavaScript, com animacoes de entrada, cards interativos e estrutura de secoes bem definida.

## Destaques

- Layout responsivo com tipografia personalizada
- Animacoes de revelacao usando IntersectionObserver
- Secoes de resumo, objetivos, habilidades, projetos, formacao e contato
- Assets locais para imagens e PDFs

## Stack

- HTML
- CSS
- JavaScript

## Estrutura

- Public/index.html - pagina principal
- Public/CSS/style.css - estilos globais
- Public/JS/script.js - animacoes e interacoes
- Public/Assets/IMG - imagens
- Public/Assets/PDF - certificados e curriculo
